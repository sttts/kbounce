# Datenschutzerklärung für KBounce

Stand: Januar 2026

KBounce ("das Spiel") wird von Stefan Schimanski entwickelt.

## Datenerhebung

Das Spiel erhebt minimale Daten für die Online-Bestenliste:
- Nickname (optional, Sie wählen selbst, was angezeigt wird)
- Spielstand und Level
- Ungefährer Standort (Land/Stadt, abgeleitet aus der IP-Adresse)
- Screenshot abgeschlossener Level (optional)
- Plattform (iOS, Android, Web, macOS, Windows, Linux)
- Betriebssystemversion
- App-Version

Diese Daten werden auf unseren Servern gespeichert, um die globale Bestenliste anzuzeigen und für Analysen.

## Nicht erhobene Daten

Wir erheben keine:
- Persönlichen Informationen (Name, E-Mail, Telefon)
- Gerätekennungen
- Nutzungsanalysen oder Tracking-Daten

## Datenspeicherung

Bestenlisten-Einträge werden unbegrenzt aufbewahrt. Sie können die Löschung beantragen, indem Sie kbounce@1stein.org kontaktieren.
